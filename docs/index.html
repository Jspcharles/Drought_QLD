Queensland Drought Data Platform

Monthly SPI and SPEI at four accumulation scales, plus a bioregion-aware Protracted Drought Classification, gridded at 0.05° across Queensland, Australia. Built for researchers, agribusiness, and anyone working on drought risk — free to explore, free to download.

What's in the data

Three complementary products, all derived from SILO gridded climate observations and consistent across the same 382 × 311 grid.

SPI

Standardized Precipitation Index

Rainfall-based drought signal, standardized against the long-term local climatology.

SPEI

Standardized Precipitation-Evapotranspiration Index

Adds atmospheric water demand to the rainfall signal, via gamma-distribution fitting.

How protracted drought is defined

A repeat drought event is only "protracted" if it strikes before a location has had time to recover — and how long that takes depends on the bioregion.

Drought onset is flagged where SPI-1 ≤ -1.0. Recovery is confirmed once SPI-1 ≥ -0.5. If a new drought begins again before that bioregion's typical recovery window has elapsed, it's classified as protracted — capturing cumulative stress that single-event drought indices miss entirely.
